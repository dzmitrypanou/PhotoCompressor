# Photo Compressor

Массовый компрессор фотографий с интерфейсом в стиле Discord.

Стек: **Electron + React + TypeScript + Sharp**.

## Возможности

- Массовое сжатие JPEG, PNG и WebP
- 5 пресетов качества: Максимум, Высокое, Сбалансированное, Веб, Агрессивное
- Параллельная обработка через worker threads (Sharp / libvips)
- Выбор исходной и выходной папки, рекурсивный обход
- Удаление EXIF, перезапись файлов, выбор формата выхода
- Прогресс, скорость, ETA и лог обработки

## Требования

- Node.js 20+
- Windows 10/11

## Установка и запуск

```bash
npm install
npm run dev
```

## Сборка .exe

```bash
npm run build
```

Установщик появится в папке `dist/`.

Только сборка приложения без installer:

```bash
npm run build:app
```

## Пресеты

| Пресет | JPEG | WebP | Макс. сторона |
|--------|------|------|---------------|
| Максимум | 92 | 90 | без изменений |
| Высокое | 85 | 82 | без изменений |
| Сбалансированное | 75 | 72 | 2560 px |
| Веб | 70 | 68 | 1920 px |
| Агрессивное | 55 | 52 | 1280 px |

## Производительность

На 8-ядерном CPU Sharp обычно обрабатывает 150–400 JPEG в секунду (зависит от разрешения и диска).

## Структура

- `electron/` — main process, IPC, Sharp worker pool
- `src/` — React UI в стиле Discord
- `shared/` — общие пресеты и утилиты
